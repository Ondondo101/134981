AliasNbPages

AliasNbPages([string alias])

Omschrijving

Definieert een alias voor het totaal aantal pagina's. Het wordt vervangen als het document gesloten wordt.

Parameters

alias
De alias. Standaard waarde: {nb}.

Voorbeeld

class PDF extends FPDF
{
function Footer()
{
    //Ga naar 1.5 cm van de onderkant
    $this->SetY(-15);
    //Selecteer Arial italic 8
    $this->SetFont('Arial','I',8);
    //Geef huidig pagina nummer en totaal aantal pagina's
    $this->Cell(0,10,'Page '.$this->PageNo().'/{nb}',0,0,'C');
}
}

$pdf=new PDF();
$pdf->AliasNbPages();
