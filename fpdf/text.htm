Text

Text(float x, float y, string txt)

Omschrijving

Geeft een string weer. Het begin is aan de linkerkant van het eerste karakter op de baseline. Deze methode staat toe een string ergens precies op de pagina te plaatsen, maar het is meestal makkelijker om Cell(), MultiCell() of Write() te gebruiken, wat standaard methodes zijn om teksten weer te geven.

Parameters

x
Abscis (=X coördinaat) van het beginpunt.
y
Ordinaat (=Y coördinaat) van het beginpunt.
txt
String om weer te geven.
